# Agent Marketplace — USER STORIES

## Discovery & install (consumer)
1. As a Consumer, I want to search agents/servers by capability, so that I find what I need. [MVP]
2. As a Consumer, I want categories + popularity sorting, so that I browse. [MVP]
3. As a Consumer, I want ratings/reviews, so that I judge quality. [MVP]
4. As a Consumer, I want one-click install into ContextOS, so that I use it in my workflow immediately. [MVP]
5. As a Consumer, I want to see what permissions/tools an item needs, so that I install safely. [MVP]
6. As a Consumer, I want security-review status shown, so that I trust it. [V1]
7. As a Consumer, I want to observe an installed item (via #4), so that I monitor it. [V1]
8. As a Consumer, I want usage-based pricing shown upfront, so that I know the cost. [V1]
9. As a Consumer, I want to try before committing, so that I evaluate. [V2]
10. As a Consumer, I want bundles/collections, so that I install related capabilities together. [V2]
