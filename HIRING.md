# Agent Marketplace — HIRING

Built in Year 3 → the team already exists (5–10+ people). New roles serve the two-sided motion. Portfolio principles: BUSINESS_STRATEGY.md §5.

- **Marketplace/platform engineer(s):** catalog, search, install, payouts (Stripe Connect) — reuse #1/#3/#4.
- **Trust & safety / review lead:** owns the security-review pipeline + takedowns + curation quality (existential for a marketplace).
- **Creator-relations / ecosystem manager:** recruits + supports creators (supply side); runs the creator program. The key marketplace-specific role.
- **Growth/marketplace ops:** manages the flywheel (supply↔demand), promotions, featured placement.
- **(Reuse) DevRel + sales:** from the portfolio team.

**Principles:** by Year 3 you hire ahead a bit (marketplace needs both sides nurtured), but only after the platform funds it. Trust & safety + creator relations are the non-obvious, essential hires. Don't staff this until the platform's distribution justifies the marketplace at all.
